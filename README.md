# Task 3: SQL for Data Analysis

## Objective
The objective of this task is to use SQL queries to extract, manipulate, and analyze data from an Ecommerce Sales Dataset.

## Dataset
Dataset Name: Ecommerce Sales Data

Columns:
- Order Date
- Product Name
- Category
- Region
- Quantity
- Sales
- Profit

## Tools Used
- MySQL Workbench
- SQL
- GitHub

## SQL Concepts Covered

### Basic Queries
- SELECT
- WHERE
- ORDER BY
- GROUP BY

### Aggregate Functions
- SUM()
- AVG()
- COUNT()
- MAX()
- MIN()

### Joins
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN

### Subqueries
- Sales above average
- Profit analysis

### Views
- Regional Sales Summary
- Product Performance View

### Query Optimization
- Created indexes on frequently filtered columns

## Key Insights

- Electronics generated the highest sales.
- North region contributed significant revenue.
- Products with sales above average were identified using subqueries.
- Regional sales and profit summaries were created using views.

## Outcome

Learned how to:

- Import CSV data into SQL database
- Perform data filtering
- Use aggregate functions
- Write subqueries
- Create views
- Optimize SQL queries using indexes
